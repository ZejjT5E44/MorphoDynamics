Object storing relevant information regarding the processing, e.g. the window size, the analyzed signal, the type of segmentation used.

Parameters

  • J (int) – number of window layers

  • I (list of int) – number of windows in each layer

  • num_time_points (int) – number of analyzed time points

  • num_channels (int) – number of analyzed channels
